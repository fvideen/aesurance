# AESURANCE

Demo [aesurance.org](https://aesurance.org)

See [devpost submission](https://devpost.com/software/aesurance) for more details

To run locally,

```
git clone https://github.com/AymericBethencourt/aesurance.git

cd aesurance

yarn install

aeproject deploy -n testnet 

cd src/frontend

yarn start
```
